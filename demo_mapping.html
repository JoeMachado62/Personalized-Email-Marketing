How It Works

  1. Upload Your CSV: Drag and drop any CSV file
  2. Review Auto-Detection: System analyzes columns and suggests mappings
  3. Customize Mappings: Use dropdowns to adjust how each column is used
  4. Process Data: System uses your mappings for enrichment

API Endpoints

POST /api/v1/mapping/analyze
  - Analyzes CSV columns and suggests mappings
  
POST /api/v1/mapping/save-mapping
  - Saves user-confirmed column mappings
  
GET /api/v1/mapping/{job_id}
  - Retrieves saved mappings for a job
